### Patron API flow

The page outlines the patron API flow, which involves browsing the menu, placing orders, and managing order fulfillment for table-side and takeaway services. The flow includes:

1. Starting with the Dashboard display
2. Menu browsing and Category navigation
3. Viewing dish information
4. Dish selection with customization options
5. Cart management
6. Checkout process including review, payment, and confirmation
7. Two option for delivery based on dining option (Tableside vs. Takeaway)
